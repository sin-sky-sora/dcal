# Django Full Cal

### start project from paiza.cloud
```
# project install
git clone https://github.com/sin-sky-sora/dcal.git
# pyenv
cd dcal
sh init.sh
source ~/.bash_profile
pyenv install 3.8.8
pyenv global 3.8.8
# pipenv
python -m pip install --upgrade pip
python -m pip install pipenv
pipenv install
# start project
pipenv run python manage.py runserver
# or
pipenv run start
```
